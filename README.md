# Featherblade

*Jekyll plugin to shave off all that thick, unruly CSS*

## Usage

1. Add `gem 'featherblade'` to your site's Gemfile and run `bundle`
2. Add the following to your site's `_config.yml`:

```yml
gems:
  - featherblade
```
